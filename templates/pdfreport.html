How to install django in virtual environment:
1) Goto the folder and open cmd
2) python –m venv myvenv
3) Select the interpreter as myvenv from command pallate
4) .\venv\Scripts\activate
5) pip install django
6) pip freeze & copy the detail
7) Create a requirement.txt file and paste the detail in it.
Settings for Templats and Static file
Settings.py:
'DIRS': ['templates'],
STATICFILES_DIRS=['static']
